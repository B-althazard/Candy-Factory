# Candy Factory Changelog

## v0.3.2
- Added snackbar toast notifications for key actions and failures.
- Compressed the header into a compact bar with a toggleable tools tray.
- Reworked mobile workstation behavior so FAB overlays run on the same tabbed editor state instead of restoring the long-form editor.
- Fixed dark-theme legibility for FAB, floating panels, modal surfaces, and option dialogs.
- Unified schema select fields across mobile and desktop with one searchable dialog-based option picker.
- Added visible lock-state controls on fields and fixed global randomize to lock randomized fields.
- Improved editor refresh behavior for preset apply/import flows.
